# Dev Documentation

## Data Formats

### Partition Object Format

```json
{
  "number": 6,
  "slot": "002",
  "start": "0000673792",
  "end": "0125827071",
  "length": "0125153280",
  "size": "0059G",
  "name": "Basic data partition"
}
```

  "mtime": "1667208116",
  "ctime": "1667208116",
  "crtime": "1666254306",
